# Ethical Hacking Task 01 – Information Gathering & Reconnaissance

## Intern Details
**Name:** Shraddha Antre

## Objective
The objective of this task is to understand the Reconnaissance phase of Ethical Hacking by collecting publicly available information about a target using passive information gathering techniques.

## Tools Used
- WHOIS Lookup
- DNS Lookup
- BuiltWith / Wappalyzer
- Security Headers
- Web Browser

## Task Performed
- Selected a public website as the target.
- Performed WHOIS lookup.
- Collected DNS records.
- Identified website technologies.
- Checked HTTP Security Headers.
- Analyzed robots.txt and sitemap.xml.
- Prepared a reconnaissance report with screenshots and research notes.

- Screenshots
- Research Notes
- README.md

## Learning Outcomes
- Understood passive reconnaissance techniques.
- Learned WHOIS and DNS enumeration.
- Identified web technologies and security headers.
- Understood the purpose of robots.txt and sitemap.xml.
- Improved documentation and reporting skills.
